## [0.1.0] - 2026-05-15

### User-Facing Changes

- Single-click Windows installer with prerequisite automation.
- Core MVP GroundTruth Local implementation for civil engineering workflows.
- Interactive AI copilot for project planning and estimation.
- Intelligent visual search for design documents and technical drawings.
- Data export to Excel and PDF formats for reporting.

### Developer-Facing Changes

- Initial project repository structure and development environment setup.
- Local SQLite database schema and project folder management service.
- FastAPI sidecar implementation for high-performance backend processing.
- Automated test infrastructure and CI/CD foundations.

Changelog Entry Format Guide
---------------------------
Each entry should be a bullet point starting with a capital letter, no trailing period.

Categories:
- Added:      for new features
- Changed:    for changes in existing functionality
- Deprecated: for soon-to-be removed features
- Removed:    for now removed features
- Fixed:      for any bug fixes
- Security:   for vulnerability fixes

Insert changelog entries into the relevant section under [Unreleased]
before cutting a release. When releasing, rename [Unreleased] to the
version number and date, then create a fresh [Unreleased] section above.
